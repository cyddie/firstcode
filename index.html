CSS Cheat Sheet

Common CSS Text Properties

Name Tag Description
Color {color :} Changes the color of the foreground text.
Text Align {text-align :} Aligns the text in an element e.g. left, right, center.
Spacing {letter-spacing :} Increase or decrease the space between characters.

Common CSS Font Properties

Name Tag Description
Font Family {font-family :} Specifies the font family for text (Times New Roman is the default).
Font Size {font-size :} Specifies the font size of text (pixels).
Font Weight {font-weight :} Specifies the weight of a font e.g. bold, normal, lighter.
Font Style {font-style :} Specifies the font style for text e.g. normal, italic, oblique.
